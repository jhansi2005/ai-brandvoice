AI Brand Tweet Generator

This project is a simple web-based tool that generates 10 on-brand tweets based on user inputs. The user provides information about a brand such as brand name, industry, campaign objective, and product details. The system then analyzes the brand voice and generates tweets that match the brand’s tone and communication style.

Features

Input form to enter brand information

Automatic brand voice analysis

Generates 10 tweets in the brand's tone

Displays a brand voice summary and tweet outputs

Simple and easy-to-use interface

Technologies Used

HTML – to create the webpage structure

CSS – to design and style the interface

JavaScript – to process inputs and generate tweets

GitHub Pages – to host the website and provide a live link

How It Works

The user enters brand details in the form.

The system analyzes the brand voice based on the inputs.

It generates a brand voice summary.

The tool then produces 10 tweets matching the brand's tone and style.
